# CP32 购物助手

CP32 同人展购物助手 — 线下同人展专属购物工具

## 功能特色

- 📖 批量 Excel 导入 / 手动录入商品
- 🛍️ 按摊位、按 IP、按 CP 三种浏览模式
- ⭐ S级优先标记与高级筛选排序
- ✏️ 购物车内直接编辑商品信息
- 💰 预算核算与购物进度追踪
- 📱 移动端优先的 Neo-Brutalism 风格 UI

## 技术栈

- React 19 + Vite 8
- Zustand (状态管理)
- Framer Motion (动画)
- Lucide React (图标)
- SheetJS (Excel 解析)

## 开始使用

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
